Documentation

Pour avoir plus de d'information, veuillez suivre le lien suivant : https://angular-google-chart.github.io/angular-google-chart/docs/latest/examples/

google-chart : représente ici l'attribut pour désigner une élément chart

myChartObject : représente les données

Important:
